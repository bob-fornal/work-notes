## Azure Administration

### Azure Resource Manager (ARM)

The orchestration layer for managing the Azure cloud.

* Azure Portal
* Azure CLI
* Azure Powershell

Cloud is made of resources; an entity managed my Azure.

* **Resource Groups**: Logical containers for grouping resources.
* **Azure Subscription**: Logical construct that groups together resource groups and associated resources. Billing unit for Azure cloud costs.

The ARM Connects to the resource provider (Azure Resource Provider).

* Microsoft.Compute
* Microsoft.Network
* Microsoft.Storage
